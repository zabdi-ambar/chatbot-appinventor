# Mi Chat Bot Personal

Aplicación móvil desarrollada con App Inventor que permite al usuario interactuar con un chatbot inteligente utilizando la API de ChatGPT. La app admite entrada por texto y voz, y reproduce las respuestas del bot en voz alta para mejorar la experiencia del usuario.

## Tecnologías utilizadas

- App Inventor (desarrollo móvil visual)
- API de ChatGPT (OpenAI)
- Servicios de voz de Google (SpeechRecognizer y TextToSpeech)
- GitHub (repositorio del proyecto)

## Funcionalidades

- Pantalla de bienvenida con ingreso de nombre
- Chat principal con respuestas inteligentes
- Entrada por voz y lectura de respuestas
- Navegación fluida entre pantallas
- Interfaz adaptada a dispositivos móviles

## Capturas

Las capturas de pantalla se encuentran en la carpeta `/capturas`.

## Documentación

El avance semanal está disponible en la carpeta `/documentacion` o `/avances`.

## Instalación

1. Descargar el archivo `.aia` desde el repositorio.
2. Importarlo en [App Inventor](https://appinventor.mit.edu/).
3. Compilar el APK desde el entorno de desarrollo.
4. Instalar en un dispositivo Android.
